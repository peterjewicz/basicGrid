Welcome To Basic Grid

Basic Grid Is the simplest CSS grid ever made, ready to be brought into any project and used straight away. Following the farmiliar 12 grid layout basic grid is easy for any skill level. For those that like the most control; basic grid comes with only the grid styles and nothing else.

For those that like a bit more basic grid also comes with the full sass files and the ability to customize. For example, changing the padding for the columns is as simple as modifying a single variable and running "grunt build"!

To start each grid is wrapper in a row. There are two row classes "row" and "row-fluid".

Row is your standarded center container that by default is fixed to a mx width of 1024px. Row-fluid is a completly fluid container, it will strech to fill its container. You're free to mix and match these on the site as you please.

Sweet Features

Responsive

Mobile First Approach ensures great display on any size screen. With the ability to change the grid at breakpoints it's easy to get the perfect layout everytime.

Customizable

Built with sass it's super easy to customize. Not happy with the padding? Maybe want to use a a different breakpoint size? No problem! A simple change to the sass is all that's required.

Light Weight

Basic grid is super lighweight and won't increase your load times. At just over 2KB compressed it's very tiny and ready to drop into any project with minimal effect.

Column Demo

Basic grid uses the farmilar 12 column layout to display content. Each column is denoted by a number 1-12 of how much space it should take; for example attaching the class column-large-6 to a div will give a 50% width of from the large and up breakpoint. Anything under the specified breakpoint will default to 100% width.

Basic Grid supports mixing, so adding "column-small-6 column-medium-9" will give a 75% width for medium and up, 50% for small, and 100% for extra small. You're free to mix any of the breakpoints to a single element in order to get exact layout you're looking for.

Breakpoints

Basic Grid comes with 3 default media queries and one for defaults. All the numbers listed below are defaults, but can easily be changed in the included sass files.

Large: col-large-* (Above 1200px)
Medium: col-medium-* (Above 992px)
Small: col-small-* (Above 768px)
Extra Small: col-xsmall-* (under 768px)

Customize Away

For those that like to tinker Basic Grid comes with the souce sass files. One of these is a "settings.scss" files, which controls many of the values listed above. This includes things like the breakpoint values, column padding, and row margin. If you're looking for something a little bit differerent than the defaults this gives you an easy way to do so.
